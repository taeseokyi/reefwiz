💧 도저 자동 조정 (올포리프)

일시dKH추세/일도징(원액 mL/일)모드

매일 1회(13시 측정 후) dKH 수준·추세로 조정 권고 계산(자동 적용 꺼짐) · 목표 7.2 dKH · 수동 설정은 몇 분 내(최대 ~5분) 도저에 적용됩니다 · 통은 50% 희석액이라 실도징(펌프가 뿜는 부피)은 원액의 2배입니다

⚙️ 자동 조정 설정·안전 한계 — 수조에 줄 수 있는 최대 영향
  • 적용 모드: 권고만(자동 적용 꺼짐, 7/6 사용자 지시) — 계산·기록만 하고 도저 설정은 바꾸지 않음. 수동 설정·목표 저장은 계속 적용됨
  • 오차(수준): 현재 dKH = 최근 유효 측정 3회의 중앙값과 목표의 차이
  • 속도(추세): 최근 7일 21회 측정의 기울기(Theil-Sen 중앙값 기울기, 튀는 값에 강함) — 지금 도징으로 dKH가 하루에 움직이는 속도
  • CO₂ 편향 제외: 새벽 CO₂ 편향 의심 측정(그래프 주황 점)은 추세·수준 계산에서 제외 — 판정은 ref 곡선 형태(평탄 지연+하강)로 자동. 플래그 조회 실패 시엔 제외 없이 종전대로 계산
  • 조정 원리: 필요한 속도(오차÷7일)에서 현재 속도를 뺀 만큼만 도징량을 변경 — 이미 목표 쪽으로 움직이고 있으면 그만큼 덜 바꿈(속도 보상)
  • 목표 접근: 오차를 7일에 걸쳐 좁힘 — dKH 변화 속도 상한 ±0.25 dKH/일 (허용 오차 ±0.5의 절반)
  • 1회 조정 폭: 현재 도징량의 최대 ±30%
  • 도징량 절대 범위: 원액 1.5~18.0 mL/일 또는 0 = 정지 (실도징 3~36 mL/일, 펌프 2000~24000ms; 정지는 lrt 0 = 무배출). 자동 조정은 1.5~18 안에서만 움직이고, 정지는 수동 설정으로만 가능
  • 미세 변경 무시(데드밴드): 원액 0.15 mL/일(200ms) 미만 차이는 유지
  • 감도 가정: 원액 1 mL/일 ≈ +0.006 dKH/일 (6/29 볼루스 55mL→+0.32 dKH 실측 유도)
  • 중단·보호: 최근 7일 유효 측정 10점 미만(CO₂ 제외 후)이거나 CO₂ 제외가 9점을 넘으면(판정기 오작동 의심) 조정 안 함 · 적용 실패 시 이전값 롤백 · 수동 설정이 있는 회차는 자동 조정 생략
